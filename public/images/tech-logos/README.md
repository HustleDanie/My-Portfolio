# Tech Stack Logos

This folder contains logo images for the technologies displayed in the Tech Stack section.

## How to Add Logos

1. **Download logo files** for each technology (preferably SVG format for best quality)
   - You can find official logos at:
     - [Simple Icons](https://simpleicons.org/) - Free SVG icons
     - [DevIcon](https://devicon.dev/) - Programming language and framework icons
     - Official brand websites

2. **Save logos** in this folder (`public/images/tech-logos/`) with the following naming convention:
   - Use lowercase and hyphens: `python.svg`, `javascript.svg`, `tensorflow.svg`, etc.
   - Supported formats: `.svg`, `.png`, `.jpg`

3. **File naming examples:**
   - Python → `python.svg`
   - JavaScript → `javascript.svg`
   - TypeScript → `typescript.svg`
   - TensorFlow → `tensorflow.svg`
   - PyTorch → `pytorch.svg`
   - AWS → `aws.svg`
   - Docker → `docker.svg`
   - etc.

4. **Logo requirements:**
   - Recommended size: 64x64px to 128x128px
   - Format: SVG (preferred) or PNG with transparent background
   - The logos will be displayed in 80-96px containers
